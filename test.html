ReplaceIt Extension Test Page

This page is designed to test the ReplaceIt Chrome extension. Use this page to verify that text replacement is working correctly.

Test Instructions

  1. Install the ReplaceIt extension
  2. Click the extension icon and add some replacement pairs
  3. For example: Replace "Hello" with "Hi" or "world" with "universe"
  4. Refresh this page to see the replacements in action

Sample Text for Testing

Hello world! This is a test page for the ReplaceIt extension.

Hello everyone, welcome to the world of text replacement!

The word "Hello" appears multiple times in this document.

You can replace any word or phrase, like "extension" or "replacement".

Common Test Cases

  • Hello - Try replacing with "Hi" or "Greetings"
  • world - Try replacing with "universe" or "planet"
  • test - Try replacing with "demo" or "example"
  • ReplaceIt - Try replacing with "TextReplacer"
